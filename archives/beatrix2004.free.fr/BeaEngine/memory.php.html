MEMORYTYPE Structure

Cette structure permet d'obtenir des informations sur le type d'accès mémoire. 

struct MEMORYTYPE {
   long BaseRegister;
   long IndexRegister;
   long Scale;
   long long Displacement;
};

Membres

BaseRegister

[out] Indique le registre de base dans l'écriture : [BaseRegister + IndexRegister*Scale + Displacement].

IndexRegister

[out] Indique le registre d'index dans l'écriture : [BaseRegister + IndexRegister*Scale + Displacement].

Scale

[out] Indique l'échelle : 1, 2, 4 ou 8.

Displacement

[out] Valeur du déplacement s'il existe dans l'écriture : [BaseRegister + IndexRegister*Scale + Displacement].
