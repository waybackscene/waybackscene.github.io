addTLS 1.0 - adding TLS easily in your apps

Here is a mini tool to add TLS callback functions in your binary coded in C or with masm32. It is for educational purpose. It could be useful for masm32 coders if they want to insert TLS callback quickly. You can add theorically 256 TLS callback functions.

Snoopy 1.1 - PE snooper

Voici la version 1.1 de mon petit snooper de PE. Ce petit outil permet de récupérer dans le bloc-notes toutes les chaines ANSI de plus de 3 caractères situées dans un .exe ou un .dll. Il suffit de faire un simple clic droit sur la cible pour lancer le snooper.
