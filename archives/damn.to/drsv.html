DAMN Release Signature Verifier v1.0.0

Due to every day increasing amount of lamers faking DAMN releases, all releases (starting from July 1, 2000) are digitally signed. You don't need to bother about the details (for those who're interested, it's DSA), just believe me that this algo can't be broken or forged at the current level of technical progress if it's implemented properly (it is).

This program is intended to help you in the situation when you got some release claiming to be made by DAMN, but you're not sure that it's really is what it claims to be. So, please follow the below simple steps:

  1. Check DAMN Release List. If the release you have claims to be out after January 1, 2000 and it's not in the release list - then it's 100% FAKE
  2. If you can see the release in our Release List, and if the release date is greater than July 1, 2000, then:
    • Unpack the release into some directory
    • Drag DAMN.NFO file, drop it into DAMN Release Signature Verifier window and look at the results
    • If the release doesn't pass verification - then it's 100% FAKE. You'd better delete it and start looking for the one really made by DAMN
    • If the release passes verification - then you can be 100% sure you have real DAMN release that is safe to use

Unfortunately, no guarantee can be given for the stuff released before July 1, 2000.

And one good tip for you: DO NOT download this verifier from any places other than http://www.damn.to
