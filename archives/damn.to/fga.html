I'm using Opera v5.x and it looks like your menu script doesn't work with it. Both menu and button are at the top of the page and don't move when I scroll. Opera and MSIE has different Javascript implementations, that is, a script must know what browser the user's using in order to do some things corectly. But by default Opera identifies itself as MSIE 5, making it impossible for scripts to distinguish between those two. The solution is to go to "Preferences / Connections" in Opera and set "Browser identification" to "Identify as Opera"
