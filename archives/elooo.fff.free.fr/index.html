22.03.06

Ajout de deux unpackers pour le PE Protector de Meat (Escargot v0.1) dans la section "Coding".
Les deux tools ont été codés en assembleur et sont téléchargeables avec les sources :

* Herisson 1.0 est un unpackeur statique. Le code source peut éventuellement interesser ceux qui se demandent comment travailler sur un fichier mappé en mémoire pour ensuite en faire un nouveau fichier que l'on extraira sur le disque dur.

* AuBeurre 1.0 est un unpackeur/debugger. J'ai tenté d'exploiter au max les possibilités des Debug Apis, afin de montrer comment se créer un debugger basique, poser des int3, des hardware breakpoints, faire du single-step, jouer avec les flags, rechercher des instructions particulières en mémoire, patcher, etc.

Pour finir, je me fous du ridicule des noms donnés à ces deux tools : les deux détruisent l'Escargot, c'est tout ce qui m'importe :P
