::  Win X 1.3 Win X est un logiciel qui permet de récupérer des informations sur un objet (que ce soit une fenêtre, un bouton, un champ texte ...) et de les modifier.
Caractéristiques :
Agrandir / réduire, afficher / cacher, selectionnable / non selectionnable, changement de la position / des dimensions / du texte / (du style)
::  Convert ASM > HTML Ce logiciel permet de convertir en HTML un listing ASM de OllyDbg.
Le principal intéret étant la coloration de la syntaxe (paramétrable). Le code HTML n'utilise pas les feuilles de style CSS par manque de motivation de ma part ... :(
Un plugin pour OllyDbg est inclus afin d'avoir accès au programme directement.
Le fonctionnement n'est pas parfait, le code est un peu pourri mais bon .... c'est mieux que rien :)
::  Générateur régions 1 Ce programme est un générateur de régions. Les régions permettent d'avoir une fenêtre non rectangulaire :)
Pour utiliser le programme il suffit de choisir la couleur à supprimer puis de charger un bitmap (bmp). Il est ensuite possible de sauvegarder le fichier au format binaire afin d'être inclus dans un programme ASM/Delphi/C++ ...
Sources Delphi incluses .... Pour un appercu, regardez les templates 3 et 4.
::  Générateur régions 2 Ce programme est également un générateur de régions. Il permet d'avoir des contrôles (type bouton) non rectangulaires :)
Pour utiliser le programme il suffit de choisir un bitmap (bmp) et de définir graphiquement des polygones correspondant aux "boutons". Il est ensuite possible de sauvegarder le fichier au format binaire afin d'être inclus dans un programme ASM/Delphi/C++ ...
Sources Delphi incluses .... Pour un appercu, regardez les templates 3 et 4.
::  Pé-euh ! 1.1 Bon et bien voila un truc qui peut être utile en reverse engineering :)
Pé-euh ! est un programme qui affiche des infos sur le PE-Header d'un fichier exécutable. En détail :
- Affichage des infos importantes du PE-Header
- Affichage des sections
- Affichage des fonctions importées (dll + APIs)
- Renseignements sur toutes les ressources
- Afficheur héxadécimal
Sources Delphi incluses ...

Quelques documents très intéressants sur le PE trouvés sur le Web sont inclus. (1.60 Mo)
::  Patch-Maker 1.2b Voila un outil pour les gens qui n'ont pas envie de programmer :p : un patch-maker.
Comme son nom l'indique, ca permet de créer un patch à partir d'un fichier original et celui modifé.
Le patch est personnalisable, il est possible de choisir une image de fond, de faire une forme non rectangulaire, de choisir les couleurs de dégradé des boutons ...
Programme réalisé pour un défi de programmation sur Forumcrack.
Sources 120% Delphi incluses ...
