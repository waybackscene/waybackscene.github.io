Le tutorial...
 
Salut à tous!!

Aujourd'hui, c'est le jour de cracker un simple crackme et de le keygener :)

Allez dans le CrackMe et entrez:
Name : ACiD BuRN
Serial : 123456

Pour ce crackme nous allons utiliser hmemcpy (dans SI: BPX hmemcpy).

Pressez le bouton et SI apparaît.
Pressez 1 fois F11 et ensuite F12 (j'ai pressé 39 fois F12... la 40ème affichait le message d'erreur).

Ensuite tracez le code avec F10 et vous arrivez ici:
xxxx:0040189C
Ce call est important donc allez y: F8

Tracez avec F10, vous voyez :
Mov EAX, [EAX] <== Faux Serial : 123456
Mov EDX, [EDX] <== Bon Serial: 00410043006900440020004200750052
taper : "d eax" pour voir le mauvais code
taper : "d edx" pour voir le bon code

Pour être sûr, il n'y a pas d'autre solution que de tracer avec F8 et entrer dans le call suivant.
Maintenant tracez avec F10 , vous voyez:
Cmp eax , edx <=== eax = 123456 et edx = 00410043006900440020004200750052
jz blablabla ... <=== jump to code cracker si eax = edx donc on a juste besoin d'entrer le serial en EDX..

Cool!! Le crackme nous dit bon job =)) et comme prix nous avons une image cool!!!

Maintenant comment faire un keygen?:

Pour cela, vous n'aurez besoin que d'un cerveau!!! hehe
Le serial pour ACiD BuRN est : 00410043006900440020004200750052
Et quelle est la valeur ASCII de ACiD BuRN ? : 4143694420427552

Donc, le crackme prend chaque valeur ASCII de votre nom et lui ajoute "00" avant !!
C'est lame et facile pour en faire un keygen maintenant!

Source du keygen en VB5 :

----------------------------------------------
For x = 1 To Len(Text1.Text)
code = Hex(Asc(Mid$(Text1.Text, x, 1)))
Text2.Text = Text2.Text & "00" & code
Next x
-----------------------------------------------
