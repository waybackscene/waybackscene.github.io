Pulsar's JunkCode
Crackme

Pulsar

Salut tout le monde,

C'est un de mes premiers crackme donc soyez indulgents svp :)

Si vous voulez apprendre de nouvelles méthodes de detecter softice / olly, de nouvelles manières pour effacer votre PE header, protéger votre IAT ... je pense que ce crackme n'est pas fait pour vous.

Par contre si vous voulez avoir la possibilité d'apprendre (ne m'engueulez pas si vous n'apprenez rien) certaines fonctionnalités de vos outils préférés (IDA me vient de tête) ou de construire certains programmes qui vous aideront sans doute un jour ou l'autre dans votre carrière de reverse, alors ces crackmes peuvent vous être utiles.

Ces crackmes, oui...en effet je vais en faire environ 5 avec un niveau de difficulté croissant. Si vous arrivez à les résoudre tous, je vous offrirai un petit cadeau :)

Pulsar's JC#1 :
Ce premier crackme est simple, vous vous apercevrez rapidement que je prends une info, je génère des trucs et après je dis si vous êtes un bon reverser ou pas... La routine de calcul elle même est courte, mais malheureusement "un peu" de junk code a infesté mes sources...Je pense que vous n'aurez aucune difficultés à trouver les mots magiques quand le code sera propre !

Pour réussir ce crackme vous devez trouver le bon code... Si vous voulez le bruteforcer, pas de problème, mais vous n'apprendrez sûrement pas grand chose... ça ne vous aidera pas pour le prochain en tout cas :)

Pulsar's JC#2 :
Bon rien de bien méchant pour ce second crackme non plus, simplement le moteur de génération s'étoffe un peu plus. Je pense cependant qu'enlever le junk de celui-ci pourrait être presque plus facile que pour le précédent...A vous de voir pourquoi :)

Pour réussir il suffit de faire afficher "Good Boy" au programme.

Info:
Je n'ai pas codé tout le junk code moi même, c'est un outil maison qui s'en est occupé. Avec les évolutions de cet outil viendront des difficultés supérieures pour les crackmes...

J'espère que vous apprendrez des choses intéressantes...
