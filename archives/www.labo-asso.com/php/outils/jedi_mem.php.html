Jedi Memory
(version 1.0.0.0)
-=¤ Lionel d'Hauenens ¤=-


Introduction

Cette dll propose des fonctions permettant de manipuler toute la mémoire virtuelle (0x00000000-0xFFFFFFFF) à partir du mode user.

Compatibilité

jedi_mem est compatible avec les systèmes fonctionnant avec des pages mémoire de 4Ko ou de 4Mo.
-> Fonctionne sous Windows 2000/XP/Server 2003 SP0
-> Ne fonctionne pas sous Windows Server 2003 SP1

note :

Pour pouvoir manipuler toute la mémoire virtuelle, jedi_mem mappe la mémoire physique en mémoire virtuelle à l'aide de l'objet \Device\PhysicalMemory.
Malheureusement, le Service Pack 1 de Windows Server 2003 bride cet objet en le rendant inaccessible à partir du mode user. Cette dll ne peut donc pas, à l'heure actuelle, fonctionner dans cet environnement. Un driver permettant de débrider l'objet \Device\PhysicalMemory sous Windows Server 2003 SP1 est en cours de développement et devrait être publié très bientôt.

Attention

La portée de cette dll dépassant la simple mémoire utilisateur, il est fortement conseillé de travailler en connaissance de cause.
