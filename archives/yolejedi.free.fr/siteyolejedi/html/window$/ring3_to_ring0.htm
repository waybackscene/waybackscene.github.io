jedi_land.dll est la nouvelle version de ma dll nommée ring0.dll. Tout le concept a changé et l'utilisation a été simplifiée. C'est pourquoi ring0.dll est devenue jedi_land.dll :)

Cette dll permet de passer en mode kernel (ring0) à partir du mode utilisateur (ring3). Le code ring0 est exécuté dans le même context que le thread ring3 original. jedi_land.dll fonctionne sous win 9x/Millennium/2000/Server2003/XP mais ne fonctionne pas sous WinNT !
