Il est parfois très utile de pouvoir détecter sous quel type de Window$ notre application tourne. Certaines API sont bien sûr prévues à cet effet mais le but de cette section est d'aller plus loin en concevant des routines indépendantes.
Au delà du résultat trouvé (qui n'est qu'un prétexte), ce domaine passionnant permet de développer une compréhension sur les différentes réactions de window$ dans un context donné. Une première approche consiste à différencier le noyau 9x du noyau NT mais des différences parfois très fines existent entre les systèmes issus du même noyau.
